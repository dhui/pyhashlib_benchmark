# pyhashlib_benchmark

Benchmarks the built-in Python hash algorithms

## Setup

1. Install [poetry](https://python-poetry.org/docs/#installation)
1. Run: `poetry install --no-dev`


## Running benchmarks

`poetry run ./benchmark.py -f ${FILE_TO_HASH}`

## Development

### Setup

`poetry install`

### MyPy

`poetry run mypy .`

### Black

`poetry run black .`
